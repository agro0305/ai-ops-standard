# AIS-0006 — Execution

## Purpose

Execution applies an approved plan while minimizing uncontrolled change.

> **AIS-0006-REQ-001:** Execution MUST match the approved plan and scope.

> **AIS-0006-REQ-002:** Preconditions, backup evidence and approval state MUST be checked immediately before the first state-changing action.

> **AIS-0006-REQ-003:** Changes SHOULD be atomic, incremental and observable.

> **AIS-0006-REQ-004:** Existing configuration MUST be edited rather than replaced wholesale unless replacement is explicitly justified.

> **AIS-0006-REQ-005:** The agent MUST record commands, API operations, changed files and service actions with secrets redacted.

> **AIS-0006-REQ-006:** Unexpected output, changed discovery state or failed preconditions MUST pause execution.

> **AIS-0006-REQ-007:** The agent MUST NOT disable authentication, authorization, firewall, safety controls or backup protections merely to complete a task.

> **AIS-0006-REQ-008:** Destructive actions MUST have explicit approval bound to the exact target and operation.

> **AIS-0006-REQ-009:** Restarts and reboots MUST be limited to the smallest required scope.

> **AIS-0006-REQ-010:** Partial completion MUST be reported accurately; the agent MUST NOT claim success before verification.
