# AIS-STYLE-0001 — AI-OPS Document Style Guide

## Purpose

Defines mandatory structure, terminology, metadata and editorial rules.

## Normative language

MUST, MUST NOT, SHOULD, SHOULD NOT, MAY and OPTIONAL are normative.

Requirement IDs use:

```text
AIS-<document>-REQ-<number>
```

Each canonical specification MUST include document ID, title, status, version, language, canonical flag, authors, date and revision history.

Allowed statuses:

- Draft
- Review
- Accepted
- Deprecated
- Superseded
- Withdrawn

English is canonical. Serbian translations use `<DOCUMENT_ID>.sr.md`.
