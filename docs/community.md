# Community

AI-OPS Standard welcomes technical feedback, implementation reports, specifications, schemas, tests and documentation improvements.

## Contributing

Read the repository contribution guide:

[CONTRIBUTING.md](https://github.com/agro0305/ai-ops-standard/blob/main/CONTRIBUTING.md)

All changes should preserve read-only discovery, dry-run defaults, explicit approval for high-risk operations and deterministic audit records.

## Support

Usage questions and reproducible bug reports are covered by:

[SUPPORT.md](https://github.com/agro0305/ai-ops-standard/blob/main/SUPPORT.md)

Sanitize logs before posting them. Do not include credentials, private hostnames, production data or customer information.

## Security

Potential vulnerabilities must follow the private reporting process in:

[SECURITY.md](https://github.com/agro0305/ai-ops-standard/blob/main/SECURITY.md)
