# AI-OPS Skill

Before any state-changing operation:

1. Discover the current state.
2. Identify existing services, ports, MCP servers and configurations.
3. Classify risk.
4. Create a rollback plan.
5. Back up every existing file that will be changed.
6. Apply the smallest possible change.
7. Verify the resulting state.
8. Produce an operation report.
